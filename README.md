 Electricity Demand Analysis & Forecasting Dashboard

This project focuses on analyzing electricity consumption data and predicting future demand using data cleaning, Excel dashboards, and forecasting techniques.
 Dataset
File: `5statesdata.csv`
Contains electricity consumption data across multiple states with date information

 Project Workflow

 1. Data Collection
Collected raw electricity consumption dataset

2. Data Cleaning (Python)
Renamed columns for better understanding

Removed empty rows

Converted date column into proper datetime format

Sorted data chronologically

Exported cleaned dataset for further analysis

 4. Data Analysis (Excel)

Imported cleaned dataset into Excel
Created Pivot Tables to summarize data
Analyzed trends across different states and time periods

 4. Dashboard Creation

Designed an interactive dashboard using Excel
Included charts for:

Electricity consumption trends
State-wise comparison
Time-based analysis

 5. Forecasting 📈

Added forecast chart in Excel
Predicted electricity demand for the next 7 days
Visualized future trends in the dashboard
